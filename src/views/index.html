Nicole's Request Header Parser Microservice

This is a microservice that allows you to get information about your request, which will be sent in JSON format

Example Usage:

https://request-header-parser.nicole-lancaste.repl.co/api/whoami

Example Output:

{"ipaddress":"::ffff:159.20.14.100","language":"en-US,en;q=0.5",
"software":"Mozilla/5.0 (X11; Ubuntu; Linux x86_64; rv:50.0) Gecko/20100101 Firefox/50.0"}
